# Space Invaders with Raylib WIP

This is a copy of space invaders written with Raylib

## Building the game

You need to have Raylib installed on your machine
It can be found from <https://github.com/raysan5/raylib/tree/master>

1. Clone repository
2. Run `gcc -o game main.c -lraylib`

## Running the game
Run `./game`
